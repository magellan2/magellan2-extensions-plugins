Das MapIcons Plugin für Magellan wird aus den Nachrichten im Report 
bestimmte Ereignisse in den betroffenen Regionen visualisieren.

Dies wird über eine kleine Information bei der entsprechenden Region 
bewerkstelligt, einem sogenannten Tag. Diese Tag nennt sich "regionicon" 
und hat als Wert den Namen einer kleiner Grafik, die dann in der Region 
angezeigt wird. Dies ist ein altes Feature von Magellan, welches auch in
Magellan 2 noch funktioniert.

Anhand der besonderen Namen für die durch dieses PlugIn verwendeten Dateien 
können die eingefügten Tags auch wieder entfernt werden, dies kann über das
PlugIn-Menu in Magellan gesteuert werden.
