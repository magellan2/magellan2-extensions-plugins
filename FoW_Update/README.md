# Fog of WAR PlugIn

## english

This PlugIn adapts the Fog of War (FoW) consistently to the information from the CR.
It is lifted from all places where sufficient information about a region exists, regardless of where the information comes from.

## deutsch

Dieses PlugIn passt den Nebel des Krieges (Fog of War, FoW) konsequent an die Informationen aus dem CR an.
Er wird über alll dort gelüftet, wo ausreichend Informationen über eine Region existieren, unabhängig davon, woher die Informationen stammen.
